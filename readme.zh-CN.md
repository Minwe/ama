# 向我发问！

## [提问](../../issues/new) &nbsp;&nbsp;&nbsp; [浏览问题](../../issues?q=is%3Aissue+is%3Aclosed)

好多人通过电子向我提问，通过这种方式任何人都可以浏览答案！

你可以提任何问题，私人问题，金钱，工作，生活，代码，都可以。

-

### 提问指南

- 提问之前请先确认问题是否已经被回答过。
- 言简意赅。
- Bug 反馈和功能需求应该放在相关项目的 issue 中。
- 对于支持问题， Stack Overflow 是个更好的去处。
- 礼貌待人 :)
